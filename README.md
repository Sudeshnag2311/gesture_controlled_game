# Gesture-Controlled Web Game using Computer Vision

This project allows controlling a web-based game using hand gestures through a webcam.

## Features
- Jump
- Move Left
- Move Right
- Move Down

## Technologies Used
- Python
- OpenCV
- MediaPipe
- PyAutoGUI

## How to Run
1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt
3. Run the program:
   python game.py
4. Open a web-based game and allow webcam access
